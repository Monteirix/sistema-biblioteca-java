# 📚 Sistema de Gerenciamento de Biblioteca

Sistema desktop desenvolvido em **Java Swing** com integração ao **MySQL**, utilizando o padrão **DAO (Data Access Object)** para separação de responsabilidades.

---

## 🚀 Objetivo do Projeto

Desenvolver uma aplicação CRUD completa para gerenciamento de livros, aplicando conceitos de:

- Programação Orientada a Objetos (POO)
- Padrão DAO
- Integração com banco de dados MySQL
- Interface gráfica com Java Swing
- Versionamento com Git e GitHub

---

## 🛠 Tecnologias Utilizadas

- Java
- Java Swing
- MySQL
- JDBC
- Git
- GitHub
- NetBeans

---

## 📌 Funcionalidades

- ✅ Cadastrar livro
- ✅ Listar livros (JTable)
- ✅ Editar livro
- ✅ Excluir livro
- ✅ Buscar livro
- 🔒 Integração com banco via DAO

---

## 🧠 Conceitos Aplicados

- Encapsulamento
- Separação de camadas (View / Model / DAO)
- Manipulação de banco de dados com PreparedStatement
- Tratamento de exceções
- Versionamento de código
